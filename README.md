# 丁元英 · 强势文化生存术

被项目割过 30 万、跟着大佬抄作业越抄越亏、35 岁被优化后开始失眠、家里张口就要 X 万救急——

这个 Skill 不是让你变厉害，是让你**少被收割**。

---

## 这是给谁的

- 在市场里挨过打、不想再当韭菜的普通人
- 总被"靠 XX 就能成"的叙事勾走、回头才发现自己是燃料的人
- 想活个明白，不想被情绪和贪嗔痴牵着走的人

不适合：想找一句话鸡汤、想速成、想被人哄着夸的人。

---

## 你能拿它干什么

打开 Claude，把下面任何一句话发过去，它会切到丁元英的视角陪你拆：

- 「我被一个项目割了 30 万，怎么复盘」
- 「跟着 XX 大佬抄作业靠不靠谱」
- 「要不要 all in 这个生意」
- 「家里要我 5 万救急，给还是不给」
- 「35 岁被裁了，下一步怎么走」
- 「想活个明白」

它会用六把刀帮你切：

1. **文化属性三层透视**——分清这事是规律、是人性、还是你自己的执念
2. **靠字警报**——只要你心里冒出「靠 XX 就能 XX」，先停下
3. **三问法**——高收益机会过滤器，三问过不了就是陷阱
4. **觉者由心生律**——戒嗔戒痴，对抗情绪化下注
5. **设计性强势 + 两道门**——必要时主动做局，但有伦理护栏
6. **静守与退场**——对抗注意力消耗，知道什么时候不动手

## 凭什么信它

不是从网上抓几篇影评拼出来的。

基于豆豆原著《遥远的救世主》36.8 万字一手语料 + 28 个权威网络源深度调研，提炼出 6 个心智模型 + 10 条决策启发式 + 完整表达 DNA。每个心智模型都标注了**它在什么情况下会失效**——这条比"它好在哪"更重要。

## 必读：它不是什么

跳过这段直接用，你会踩到 Skill 反对的所有陷阱：

1. **丁元英是虚构人物**——豆豆笔下的文化符号，不是真实存在的人
2. **「父亲拔氧气管」原著没有**——是 2008 电视剧《天道》第 3 集加的戏
3. **「见路不走」不属于本书**——那是《天幕红尘》叶子农的概念
4. **故事窗口 1995-1998**——那套打法离开特定红利窗口几乎无法复制
5. **丁元英本人有 6 处明确的言行不一**——他自承"我这么做已经很不要脸了"，不是无矛盾的圣人
6. **完全模仿"冷处理 + 不解释"会让你失去人际关系**——他付得起代价，普通人未必
7. **设计性强势是双刃剑**——使用前必过两关：动机不能是贪嗔痴 + 你扛得起骂名
8. **本 Skill 不替代你自己思考**——「自己觉到悟到的，才是自己的」。Skill 是镜子不是救主
