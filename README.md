# 丁元英 · 强势文化生存术 Skill

> 「强势文化造就强者，弱势文化造就弱者，这是规律，也可以理解为天道，不以人的意志为转移。」

一个 Claude Code / Claude Skill 系统的人物思维框架插件——把豆豆小说《遥远的救世主》主角丁元英的思维方式蒸馏成可调用的 AI 思维顾问。

**定位**：普通人在市场中生存与逆袭的认知工具——用强势文化框架识别陷阱、戒贪嗔痴、按规律办事、必要时设计性出手。

---

## 它是什么

不是一个聊天机器人扮演丁元英陪你聊天，而是一套**可运行的认知操作系统**，帮你：

- 用「文化属性」三层透视任何人/事/局
- 用「靠字警报」识别救主叙事陷阱
- 用「三问法」过滤高收益机会
- 用「觉者由心生律」戒嗔戒痴
- 必要时按「设计性强势」做局，且自带"两道门"伦理护栏
- 用「静守与退场」对抗注意力消耗

底层是 6 个心智模型 + 10 条决策启发式 + 完整表达 DNA + 10 条诚实边界。

## 为什么用 6 个心智模型而不是 10 个、20 个

调研阶段（详见 `references/research/`）从原著 36.8 万字中梳理出 10 个系统性论点 + 6 处言行不一 + 8 类即兴反应模式。Phase 2 提炼时按"三重验证"标准（跨域复现 / 生成力 / 排他性）筛到 6 个——再多就是堆砌，再少则会丢失"设计性强势"和"静守与退场"这两个不可或缺的循环节点。

每个心智模型都标注了：
- 一句话核心
- ≥2 个原著章节出处的直引证据
- 普通人怎么用
- 这个模型在什么情况下会失效（**这条最重要**）

## 它**不是**什么——必读的诚实边界

如果跳过这一段直接用，你会踩到 Skill 反对的所有陷阱：

1. **丁元英是虚构人物**——豆豆笔下的"文化符号"，不是真实存在的人
2. **「父亲拔氧气管」原著没有**——是 2008 电视剧《天道》第 3 集添的戏。原著中丁元英对家人是克制内疚型，不是冷酷反传统型
3. **「见路不走」不属于本书**——那是《天幕红尘》叶子农的概念
4. **故事时间窗 1995-1998**——这套打法离开那个红利窗口几乎无法复制
5. **丁元英有 6 处明确的言行不一**——他不是无矛盾的圣人。他自承"我这么做已经很不要脸了"
6. **完全模仿"冷处理 + 不解释"会让你失去人际关系**——他付得起代价，普通人未必
7. **设计性强势是双刃剑**——使用前必过两关：（1）动机不能是贪嗔痴；（2）后果你扛得起骂名
8. **本 Skill 不替代你自己思考**——「自己觉到悟到的，才是自己的」。Skill 是镜子不是救主

完整 10 条边界见 `SKILL.md` 内「诚实边界」章节。
